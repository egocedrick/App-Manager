# App Manager

## Overview
Android utility for uninstalling and reinstalling apps with guided dialogs and APK scanning.

## Current Features
- Auto-detect installed apps
- Uninstall/reinstall buttons
- APK scanning in default folder
- AlertDialog confirmations

## Pending
- Folder tracing logic refinement
- Configurable folder path
- Improved uninstall targeting

## Limitations
- Hardcoded folder path may cause errors
- Uninstall logic not yet fully polished
